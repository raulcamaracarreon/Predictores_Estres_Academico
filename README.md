# Predictores_Estres_Academico
Análisis end-to-end sobre predictores de estrés académico. Se demuestra empíricamente que un modelo parsimonioso de 4 variables iguala el rendimiento ($R^2 \approx 0.88$) de modelos complejos. Incluye EDA, Ridge Regression con GridSearch, validación cruzada y un Paper de investigación generado por IA.
